## Package Shape

```tsx
import { Button, Card, PhenixProvider } from 'phenix-ui'

export function App() {
  return (
    <PhenixProvider>
      <Card.Root>
        <Card.Body>
          <Button colorPalette="ember">Review case</Button>
        </Card.Body>
      </Card.Root>
    </PhenixProvider>
  )
}
```

Phenix UI follows Chakra UI v3 conventions: compound components, semantic tokens, `colorPalette`, and provider-based theming.
