# Typography

Phenix UI typography is designed for scanning, decision-making, and sustained reading. It uses a restrained scale with clear hierarchy.

## Writing With Type

- Use one page title per page.
- Keep button labels short and verb-led.
- Use compact text only when density helps the task.
- Do not rely on size alone for hierarchy; pair type with structure and spacing.
