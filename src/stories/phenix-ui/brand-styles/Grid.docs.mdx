The Phenix UI grid keeps dense product screens readable across breakpoints. Use it for page structure, not for every small alignment decision.

## Rules

- Start mobile layouts with one column.
- Use two columns only when the relationship between regions is clear.
- Keep line lengths readable in content-heavy pages.
- Align major page regions to shared grid columns.
