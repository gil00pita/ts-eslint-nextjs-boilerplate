# Motion

Motion in Phenix UI helps people understand cause, location, and state. It should be quick, subtle, and interruptible.

## Rules

- Motion should reveal relationship, not attract attention for its own sake.
- Non-essential motion must respect reduced-motion preferences.
- Loading motion should communicate activity without suggesting precise progress.
- Avoid movement that shifts layout after people start reading.
