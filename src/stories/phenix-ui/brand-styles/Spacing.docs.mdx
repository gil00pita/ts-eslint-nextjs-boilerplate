# Spacing

Spacing is based on a 4px rhythm. Product screens should feel compact but breathable, especially around forms and tables.

## Scale

| Token | Value | Common use                         |
| ----- | ----- | ---------------------------------- |
| `1`   | 4px   | Tight icon and text gaps.          |
| `2`   | 8px   | Compact control spacing.           |
| `3`   | 12px  | Related label and content spacing. |
| `4`   | 16px  | Default component padding.         |
| `6`   | 24px  | Section spacing.                   |
| `8`   | 32px  | Page region spacing.               |
| `10`  | 40px  | Major vertical separation.         |
| `12`  | 48px  | Large editorial spacing.           |

## Rules

- Use `gap` for Stack, HStack, and VStack layouts.
- Keep related controls closer than unrelated sections.
- Use larger spacing to signal a new task or region.
- Avoid one-off pixel values unless a component requires exact geometry.
