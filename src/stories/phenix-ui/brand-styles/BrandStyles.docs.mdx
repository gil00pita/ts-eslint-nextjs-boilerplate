# Brand Styles

Phenix UI brand styles create a practical visual system for interface-heavy products. The identity is warm, crisp, and durable: enough character to be recognizable, but restrained enough for long work sessions.

## Visual Personality

Phenix UI should feel:

- Focused, not severe.
- Warm, not playful.
- Modern, not trend-led.
- Efficient, not cramped.
