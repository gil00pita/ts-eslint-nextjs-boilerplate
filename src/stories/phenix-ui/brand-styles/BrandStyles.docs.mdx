# Brand Styles

Phenix UI brand styles create a practical visual system for interface-heavy products. The identity is warm, crisp, and durable: enough character to be recognizable, but restrained enough for long work sessions.

## Foundations

| Foundation           | Purpose                                                      |
| -------------------- | ------------------------------------------------------------ |
| Available Components | Shows the current component inventory and maturity level.    |
| Color                | Defines palettes, semantic roles, and contrast expectations. |
| Icons                | Explains icon sizing, stroke weight, and accessibility use.  |
| Illustrations        | Sets the tone for product and empty-state artwork.           |
| Motion               | Keeps state changes understandable and respectful.           |
| Grid                 | Provides layout structure across breakpoints.                |
| Spacing              | Creates rhythm using predictable token steps.                |
| Radius               | Sets shape language for controls, panels, and surfaces.      |
| Themes               | Documents light, dark, and high-contrast behavior.           |
| Typography           | Defines type scale, hierarchy, and reading patterns.         |

## Visual Personality

Phenix UI should feel:

- Focused, not severe.
- Warm, not playful.
- Modern, not trend-led.
- Efficient, not cramped.
