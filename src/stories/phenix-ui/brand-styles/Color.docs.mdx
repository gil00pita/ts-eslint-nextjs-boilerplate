<Callout title="Usage Rules" tone="tip">
- Use semantic tokens before raw palette steps.
- Reserve Ember for primary actions and branded highlights.
- Pair every status color with text or an icon; never rely on color alone.
- Check contrast in light, dark, and high-contrast themes.
