<VStack gap={4} wrap="wrap" mb={8}>
  **brand-x-chakra** is a React component library that applies a [Brand
  X](https://example.com)-style skin on top of [Chakra UI](https://www.chakra-ui.com). It is not a
  replacement for Chakra. The point is to keep Chakra's composability, theme system, and React
  ergonomics, while shipping a set of opinionated components and tokens that feel closer to the
  Brand X Design System. Current Chakra version **3.36.0**.
</VStack>
