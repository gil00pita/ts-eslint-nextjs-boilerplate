<Callout title="Component Expectations" tone="warning">
  <ul>
    <li>Icon-only buttons must include an accessible name.</li>
    <li>
      Form inputs should use <code>Field.Root</code> so labels, help text, and errors are connected.
    </li>
    <li>Dialogs, menus, popovers, and selects should use their full compound structure.</li>
    <li>Status cannot be communicated by color alone.</li>
  </ul>
</Callout>

<Callout title="Review Moments" tone="tip">
  <p>Run accessibility checks when:</p>
  <ul>
    <li>A new workflow is released.</li>
    <li>A component is added or changed.</li>
    <li>A route introduces custom keyboard behavior.</li>
    <li>Tokens or themes are updated.</li>
  </ul>
</Callout>
