## Recommended First Steps

1. Install `phenix-ui` and its required peer dependencies.
2. Add `PhenixProvider` at the app root.
3. Import global fonts if your app does not already provide them.
4. Replace one shared pattern, such as a page header, alert, empty state, or form section.
5. Check keyboard navigation, focus states, and color contrast before rolling out more components.

## First Component

```tsx
import { Button, PageHeader, StatusBadge } from 'phenix-ui'

export function AccountPageHeader() {
  return (
    <PageHeader.Root>
      <PageHeader.Content>
        <PageHeader.Eyebrow>Account</PageHeader.Eyebrow>
        <PageHeader.Title>Renewal review</PageHeader.Title>
        <PageHeader.Description>
          Check outstanding actions before confirming the account renewal.
        </PageHeader.Description>
      </PageHeader.Content>
      <PageHeader.Actions>
        <StatusBadge colorPalette="moss">Ready</StatusBadge>
        <Button colorPalette="ember">Continue</Button>
      </PageHeader.Actions>
    </PageHeader.Root>
  )
}
```
